Displays the official exchange rate of the dollar, euro and real with the Argentine peso

== Description ==

Include in your site quotes the dollar, euro, real with the Argentine peso

The plugin makes reading xml www.dolar-bluehoy.com/api.php, a website that updates your quote data every 5 minutes.

el plugin muestra el enlace de la web www.dolar-bluehoy.com en Widgets.


== Installation ==

###1.Extract and upload the plugin###

Extract all files from the ZIP file, **keeping the file/folder structure intact**, and then upload the `cotizacion/` it to `/wp-content/plugins/`.

###2.Activate the plugin###

Go to the admin area of your WordPress install and click on the "Plugins" menu. Click on "Activate" for the "Cotizacion" plugin.

###3.Widget usage###

1. In the WordPress admin area go to "Appearance", "Widgets".
1. Drag and drop the widget "Cotizacion" to the sidebar on the right to activate it.

== Frequently Asked Questions ==

= I follow all the steps to install the plugin and activate the widget but the numbers aren't showing at all =

It's possible that there is a connection problem to the Central Bank of Argentina web service. Just try later.
